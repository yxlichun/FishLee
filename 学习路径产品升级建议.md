# AI学习路径产品升级建议

> 从"AI PM学习路径"（单用户、单路径、无AI）升级为通用的AI辅助学习路径产品

---

## 一、核心抽象：从"一条固定路径"到"任意路径"

当前最大的限制是学习路径硬编码在 `learningPath.ts` 里。要通用化，第一步是让路径变成**用户可创建、可编辑的数据**。

**具体改动：**
- `Phase/Section/Task/Resource` 从静态数据变成跟 `Note/Plan` 一样的可持久化实体
- 新增"路径编辑器"页面：用户可以创建自己的学习路径（比如"考PMP"、"学Rust"、"转行数据分析"）
- 支持多条路径并行，Dashboard按路径分别展示进度

**AI加持的想法：** 用户输入一句话目标（"我想在3个月内学会Python做数据分析"），AI自动生成一条结构化的学习路径——包括阶段划分、每阶段的学习任务、推荐资源、时间估算。用户可以在此基础上调整。

---

## 二、AI能力引入：三个层次

### 层次一：AI生成学习路径（最核心的差异化）

这是产品的核心卖点——**不是一个空白的学习计划模板，而是AI根据你的目标、背景和可用时间，帮你"定制"一条学习路径**。

- 用户输入：学习目标 + 当前基础 + 每周可用时间 + 期望周期
- AI输出：结构化的Phase→Section→Task路径，包含时间分配、推荐资源、里程碑检查点
- 用户可以在生成结果上自由编辑调整

这直接命中了抖音分析中发现的"路径焦虑"——大家最想要的就是"一个现成的、可以直接跟着走的学习计划"。

### 层次二：AI学习助手（日常陪伴）

- **智能打卡总结**：打卡时只写几个关键词，AI帮你扩展成结构化的学习记录
- **笔记增强**：写完笔记后，AI帮你生成摘要、提炼关键概念、关联到学习路径中的对应阶段
- **每周回顾**：AI根据你这周的打卡、笔记、任务完成情况，生成一份"学习周报"——进度如何、哪些地方停滞了、建议接下来聚焦什么
- **资源推荐**：基于当前学习阶段和笔记内容，AI推荐下一步应该看的资源

### 层次三：AI学习教练（更深层）

- **知识检测**：学完一个阶段后，AI根据学习内容出几道问题，检测掌握程度
- **卡点诊断**：如果某个阶段进度停滞超过X天，AI主动分析可能的原因并给出建议（是不是任务拆得太粗了？是不是缺少前置知识？）
- **学习风格适配**：根据用户的打卡频率、笔记风格、学习时长分布，AI调整推荐策略

## 三、功能层面的升级

### 必做（通用化的基础）

| 改动 | 说明 |
|------|------|
| **多路径支持** | 一个用户可以同时有多条学习路径，侧边栏可切换 |
| **路径模板市场** | 提供一批预设路径模板（学Python、学设计、考PMP等），用户可一键导入 |
| **路径分享** | 用户可以把自己的路径导出为链接或JSON，其他人可导入 |
| **用户系统** | 从单用户本地存储升级为多用户账号系统（可以用Supabase或Firebase） |

### 增值（差异化竞争力）

| 改动 | 说明 |
|------|------|
| **AI路径生成器** | 核心卖点，如上所述 |
| **学习仪表盘升级** | 增加"学习效率趋势"（每天学了多久、效率是上升还是下降）、"坚持天数排名"等 |
| **移动端适配** | 当前是桌面端设计，打卡和笔记场景在移动端更高频 |
| **提醒系统** | 基于学习计划，到点提醒用户该学习了（可接微信/飞书通知） |

## 四、技术架构升级建议

| 层面 | 当前 | 建议 |
|------|------|------|
| 存储 | Vercel Blob单文件JSON | Supabase/PlanetScale（结构化数据库，支持多用户） |
| 认证 | 无 | Supabase Auth 或 NextAuth（支持微信登录） |
| AI集成 | 无 | 后端调用大模型API（Claude/GPT），前端流式展示 |
| 部署 | Vercel | 保持Vercel，边缘函数处理AI请求 |

---

## 五、优先级排序

### 第一期（2-3周）：让路径"活"起来

- 路径从硬编码变成可创建/编辑的数据
- 多路径支持
- 3-5个预设路径模板

### 第二期（2-3周）：引入AI核心能力

- AI路径生成器（输入目标→输出结构化路径）
- AI打卡总结、笔记增强
- 用户系统（多用户）

### 第三期（持续迭代）：深度AI + 增长

- AI学习周报
- 路径分享/模板市场
- 移动端适配
- 社区功能（可选）

## 待确认问题

1. **你现在的用户只有你自己，还是已经有其他人在用？** 这决定了多用户系统的优先级
2. **AI路径生成器这个方向你感兴趣吗？** 这可能是最强的差异化点——市面上有很多"学习追踪工具"，但"AI帮你规划学习路径"的产品很少
3. **你打算投入多少时间？** 是下班后的side project，还是会投入比较大的精力
